## Setup

- Install [Git](https://git-scm.com/), [Docker](https://www.docker.com/), [Node v18](https://nodejs.org/en/download/) and [Yarn](https://yarnpkg.com/)
- Clone this repository
- Create a copy of [.env-template](./.env-template) into `.env`
- Run:
  ```
  docker compose up
  ```
- You're ready to go!
